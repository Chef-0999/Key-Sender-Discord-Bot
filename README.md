# Key-Sender-Discord-Bot

Using this discord bot, you can easily send keys or anything to your users. 


To run:

-Enter your info into the index.js file

-Npm install to install all dependencies

-node index.js 

if it says "online" then your bot is running.

**Note:**
-If the bot goes offline, all keys saved will be deleted
